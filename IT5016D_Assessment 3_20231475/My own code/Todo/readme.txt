# To-Do List Project

This is a simple to-do list application written in Python. It allows you to add, remove, and view tasks in your to-do list.

## Usage

1. Run `todo.py`.
2. Choose from the available options to add, remove, or view tasks in your to-do list.

## Options

- 1: Add task
- 2: Remove task
- 3: Show tasks
- 4: Exit
