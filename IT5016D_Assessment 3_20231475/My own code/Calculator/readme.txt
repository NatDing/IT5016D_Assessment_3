# Calculator Project

This is a simple calculator program written in Python. It can perform basic mathematical operations such as addition, subtraction, multiplication, and division.

## Usage

1. Run `calculator.py`.
2. Follow the on-screen instructions to select the operation and enter the numbers.
3. The program will display the result of the chosen operation.

## Operations

- 1: Addition
- 2: Subtraction
- 3: Multiplication
- 4: Division
